# Restaurant Allora

Restaurant Allora е ASP.NET Core MVC приложение за ресторант с онлайн меню, поръчки, резервации, любими ястия, ревюта и администраторско табло.

## Основни възможности

- Регистрация, вход, забравена парола и роли: Admin, Employee, Customer.
- Публично меню с категории, търсене и филтър по алергени.
- Клиентско меню с количка и финализиране на поръчка.
- Резервации на маси с проверка за заетост и статуси.
- Любими ястия и ревюта от клиентите.
- Админ табло с бърз преглед на поръчки, резервации, приходи и популярни ястия.
- Управление на ястия, категории, маси, поръчки, потребители и ревюта.
- Custom Bootstrap потвърждения, toast съобщения и български validation текстове.

## Технологии

- ASP.NET Core MVC
- Entity Framework Core
- ASP.NET Core Identity
- SQL Server
- Bootstrap
- Cloudinary за качване на изображения
- xUnit тестове

## Стартиране

1. Провери connection string-а `DefaultConnection` в `RestaurantAlloraProjectWeb/appsettings.json`.
2. Провери Cloudinary настройките, ако ще качваш снимки.
3. Стартирай миграциите и приложението:

```powershell
dotnet run --project .\RestaurantAlloraProjectWeb\RestaurantAlloraProjectWeb.csproj
```

При стартиране приложението изпълнява миграциите и seed-ва ролите, администраторския профил и началното меню автоматично.

## Тестове

```powershell
dotnet test .\RestaurantAllora.sln
```

## Seed по подразбиране

Проектът създава роли `Admin`, `Employee` и `Customer`, както и администраторски профил:

- Потребителско име: `admin`
- Имейл: `admin@gmail.com`
- Парола: `Admin1*`

Seed-ът за менюто добавя богати категории, ястия, ресторантски описания, алергени и Cloudinary fetch URL-и за снимките, без ръчно въвеждане на `ImageUrl` от админ формите.

Следващата планирана стъпка е разширено seed-ване на останалите таблици: примерни клиенти, служители, резервации, поръчки, любими ястия и ревюта.
